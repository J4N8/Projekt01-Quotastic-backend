## Available Scripts

In the project directory, you can run:

### Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

### Prettier

- `npm run code:format` - Format code.

### eslint

- `npm run lint:fix` - Fix some of the errors.

Run both **prettier** and **eslint** with: `npm run format`
